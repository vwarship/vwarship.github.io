iOS App 创建图标

生成App图标

  1. 下载App Icon Set Creator

  2. 运行App Icon Set Creator，拖拽图片，选择平台和输出目录，单击Generate Asset Catalog。

  3. 在Xcode中打开App，删除AppIcon，导入刚刚生成的图标集。
